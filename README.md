# 🛂 Passport Generator App

A simple web-based application that allows users to generate a passport-style preview by entering their personal details. This project focuses on frontend development and dynamic UI rendering.

---

## 🚀 Features

- User-friendly form to enter passport details
- Real-time passport preview generation
- Responsive design for all devices
- Clean and modern UI
- Dynamic data rendering using JavaScript

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
